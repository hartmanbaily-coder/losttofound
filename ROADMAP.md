# Roadmap

## Phase 1: MVP

- Private user accounts
- Custody matter setup
- Simple recurring exchange schedule
- Exchange logging
- Date-based notes
- Evidence uploads/private storage
- Child support due dates and payment records
- Basic expense tracking
- Calendar/list/timeline
- Charts
- PDF/CSV reports
- Basic audit logs
- Security/privacy documentation

## Phase 2: Security and privacy hardening

- Full MFA
- Session management page
- Login notifications
- Stronger audit trails
- More complete version history
- Encrypted field-level storage for highly sensitive fields
- Production malware scanning
- Automated dependency scanning
- Automated secrets scanning
- Backup restoration testing
- Incident response playbook
- External security review
- Privacy/legal review

## Phase 3: Advanced custody calendar

- Holiday schedules
- Summer schedules
- Rotating weekends
- School-year schedules
- Temporary order changes
- Make-up parenting time
- Manual court-order exceptions
- Multiple exchange locations
- Transportation responsibility tracking

## Phase 4: Evidence vault

- Evidence index
- Original-file preservation
- File hash/checksum
- Upload timestamp verification
- Redacted-copy creation
- Report inclusion controls
- Evidence packet builder
- Stronger evidence metadata
- Private attorney sharing

## Phase 5: Attorney/lawyer portal

- Invite attorney
- Read-only access
- Case-scoped permissions
- Revocable access
- Attorney comments
- Attorney-selected report items
- Export packet review
- Full access audit trail

## Phase 6: Advanced child support and expense tools

- Multiple support orders
- Support modifications over time
- State agency payment import by manual CSV upload
- Reimbursement request tracking
- Medical expense splits
- Childcare expense splits
- Payment reminders
- Overdue record summaries
- More advanced charts

## Phase 7: AI summaries

- AI summaries only from user-selected records
- Neutral tone
- Citations back to source entries
- No legal conclusions
- No court outcome predictions
- Attorney-summary draft
- Timeline draft
- Report cleanup assistant
- Privacy review before enabling AI on sensitive records

## Phase 8: Mobile/PWA

- Mobile-first logging
- PWA install
- Quick exchange logging
- Offline draft mode
- Secure device/session controls
- Optional discreet app name/icon

## Phase 9: Integrations

- Google Calendar export
- Apple Calendar export
- Manual email import
- Manual SMS screenshot organization
- Receipt upload
- School/daycare document upload
- Attorney document export
- No automatic inbox scraping until privacy/security review

## Phase 10: Enterprise/legal professional tools

- Attorney firm dashboard
- Strict role-based permissions
- Client-managed sharing
- Billing integration only after review
- Advanced report templates
- Jurisdiction-specific templates only after legal review
